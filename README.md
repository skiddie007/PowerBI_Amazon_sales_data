# PowerBI_Amazon_sales_data
This is an Amazon Sales Report dashboard created in Power BI, providing insights into sales performance across various categories and sub-categories. The report includes visual representations such as bar charts for sub-category sales, pie charts for category-wise profit, sales, and quantity sold, and a geographic map showing state-wise sales distribution. Key highlights include the dominance of categories like Phones and Chairs in sales and a breakdown of profit contributions.
